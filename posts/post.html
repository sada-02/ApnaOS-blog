Initial Boot & Kernel Print

Posted on April 18, 2025

After days of debugging GRUB and linking issues, we finally saw our first text printed from the kernel!

The linker script was a pain, but here's a snippet that worked:

SECTIONS {
  . = 0x100000;
  .text : { *(.text*) }
  .rodata : { *(.rodata*) }
  .data : { *(.data*) }
  .bss : { *(.bss*) }
}
