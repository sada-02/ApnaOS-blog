About ApnaOS

ApnaOS is a teaching operating system written from scratch in C. It boots via GRUB, initializes its own kernel, and provides core services that every OS needs.

  • Kernel Initialization: Handles bootloader handoff, sets up segment descriptors, and enters protected mode.
  • Process Management: Implements a round-robin scheduler and context switching between multiple processes.
  • Memory Management: Uses paging for virtual memory, with simple page allocation and free lists.
  • File System: A basic VFS layer supporting file creation, deletion, and read/write operations.
  • Shell (CLI): A command-line interface for launching programs and interacting with the system.

Each subsystem is modular, making it easy to extend features or swap in new algorithms as you learn.
